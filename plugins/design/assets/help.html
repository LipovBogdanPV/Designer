Підказки та гарячі клавіші

Вибір секцій
Клік по блоку — зробити його активним.
Переміщення блоків
Потягнути за іконку ⠿ — перемістити блок.
Натиснута Ctrl під час перетягування — вставити усередину іншого блоку.
Зміна розміру
Потягнути за край блоку — змінити ширину / висоту.
Клавіатура
Del / Backspace — видалити вибраний блок.
Ctrl+S — зберегти як шаблон (якщо увімкнено). - Не працює
